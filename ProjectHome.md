This Google Code project hosts the programs in Java that I developed for my CMSC 291 (Advanced Image Processing) class at the [Institute of Computer Science](http://ics.uplb.edu.ph), [University of the Philippnes Los Banos](http://www.uplb.edu.ph). Enjoy!

Features:

  * SET Form Processing

  * Image Enhancement

  * Plate Number Localization

  * Edge Detection

  * Morphological Operations

  * Content-based Image Retrieval

  * Motion Detection

  * Filters

  * 3D View of Image Color Distribution (RGB)
